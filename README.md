# 🧘‍♀️ Yog Sadhna Kendra - NGO Website

Welcome to the official web platform for **Yog Sadhna Kendra**, a nationwide NGO dedicated to promoting the ancient practice of Yoga for physical, mental, and spiritual well-being across India. This website is a modern, responsive, and accessible platform built to serve patients, yoga enthusiasts, volunteers, and seekers alike.

## 🚀 Tech Stack

This project is developed using the modern frontend stack:

- ⚡ [**Vite**](https://vitejs.dev/) – Fast build tool for frontend development
- 🧠 [**TypeScript**](https://www.typescriptlang.org/) – Static typing for better scalability
- ⚛️ [**React**](https://reactjs.org/) – Component-based UI library
- 🎨 [**Tailwind CSS**](https://tailwindcss.com/) – Utility-first CSS framework
- 🧩 [**shadcn/ui**](https://ui.shadcn.com/) – Accessible and elegant UI components

## 🛠️ Installation & Development

### Prerequisites

- Node.js (v16 or above)
- npm or yarn

### Steps

```bash
# Clone the repository
git clone https://github.com/your-username/yog-sadhna-kendra.git

# Navigate into the project folder
cd yog-sadhna-kendra

# Install dependencies
npm install

# Start the development server
npm run dev

npm run build
